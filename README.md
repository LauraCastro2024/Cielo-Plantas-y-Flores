Cielo Plantas y Flores 🌿

Proyecto de un Sitio web de Cielo Plantas y Flores, floristería y emprendimiento familiar de Uruguay.

Qué incluye

Catálogo de plantas y flores con filtros por categoría y buscador
Ficha de cada planta: precio, disponibilidad y cuidados (luz, riego, apta para mascotas)
Carrito de pedido que genera el mensaje de WhatsApp automáticamente
Consejos de cuidado, horarios y datos de contacto

Cómo actualizar el contenido

Todo el sitio está en un solo archivo: index.html.
Plantas, precios y stock: editar el array PLANTAS dentro del <script> al final del archivo.
Número de WhatsApp: cambiar la constante WHATSAPP (formato: 598 + número sin el 0 inicial).
Horarios y dirección: buscar la sección id="contacto" en el HTML.


Próximos pasos previstos: conectar el catálogo a una base de datos y agregar un panel de administración para gestionar stock y pedidos sin tocar código.
